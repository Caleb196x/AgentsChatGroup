Agent ChatGroupは**チャットセッション**を通じて複数のAIコーディングエージェントがプロジェクトで協力できるようにします - AIチームメンバーを調整して複雑なタスクを完了するコラボレーションワークスペースです。

## チャットセッションとは？

チャットセッションは、あなたと複数のAIエージェントがコーディングタスクで協力するグループチャットのようなものです。各セッションは専用のワークスペースであり：

- シンプルな@メンションで異なるAIエージェントにタスクを割り当て
- エージェントは各自のワークスペースで作業し、報告
- すべての会話とコード変更が一箇所で追跡

## コアバリュー

従来のコーディングエージェントは孤立して作業します - 1つのエージェント、1つのタスク。Agent ChatGroupはこれを変えます：

- **チームコラボレーション**：異なるエージェントに異なる役割を割り当て（ライター、エンジニア、コーダーなど）
- **並列作業**：複数のエージェントがプロジェクトの異なる部分で同時に作業可能
- **知識共有**：エージェントが互いの作業を参照し、以前の出力に基づいて構築

## ユースケース

チャットセッションは分業が有益な複雑なタスクに最適です：

| タスクタイプ | 役割例 |
|-----------|---------------|
| ドキュメント作成 | ライターがコンテンツ担当、エンジニアが技術詳細担当 |
| 機能開発 | コーダーが実装担当、レビュアーがコード品質担当 |
| プロジェクトセットアップ | コーダーがスキャフォールディング担当、ダウンローダーがアセット担当 |
| 調査と実装 | エクスプローラーが調査担当、コーダーが実行担当 |

<Tip>
シンプルな2エージェント設定から始めてワークフローに慣れ、必要に応じてチームを拡大してください。
</Tip>
