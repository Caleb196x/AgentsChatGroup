## AgentsChatGroupとは？

**AgentsChatGroup**は革新的な**ワークグループチャット**モードを導入し、複数のAIエージェントが本物のチームメンバーのように効率的に協力できるようにします。

- 1つのプロジェクト = 1つのグループチャット
- 異なる役割を持つ任意の数のAIメンバーを追加
- 同じ会話コンテキストですべてのエージェントを指揮
- エージェント同士が自動的に協力・コミュニケーション
- あなたはプロジェクトリーダーとしてAIチームを統括
