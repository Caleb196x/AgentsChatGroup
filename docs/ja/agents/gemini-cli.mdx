---
title: "Gemini CLI"
description: "Google Gemini CLIをセットアップ"
icon: /logos/gemini-logo.svg
---

<Steps>
<Step title="Gemini CLIを実行">
  ```bash
  npx -y @google/gemini-cli
  ```
</Step>

<Step title="ログイン手順に従う">
  プロンプトに従って認証フローを完了します。詳細は[geminicli.com](https://geminicli.com/)を参照してください。
</Step>

<Step title="agents-chatgroupを起動">
  認証完了後、agents-chatgroupを起動：

  ```bash
  npx agents-chatgroup
  ```

  タスク試行を作成する際にGemini CLIを選択できるようになりました。
</Step>
</Steps>
