<Steps>
<Step title="Run Kimi Code">
  Mac OS and Linux:
  ```bash
  # Linux / macOS
  curl -LsSf https://code.kimi.com/install.sh | bash
  kimi
  ```
  Windows:
  ```powershell
  # Windows (PowerShell)
  Invoke-RestMethod https://code.kimi.com/install.ps1 | Invoke-Expression
  kimi
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted. For more details, see the [official Kimi documentation](https://moonshotai.github.io/kimi-cli/en/guides/getting-started.html).
</Step>

<Step title="Start agents-chatgroup">
  Once authenticated, launch agents-chatgroup:

  ```bash
  npx agents-chatgroup
  ```

  You can now select Kimi Code when creating task attempts.
</Step>
</Steps>
