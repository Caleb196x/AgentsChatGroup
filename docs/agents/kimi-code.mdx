title: "Kimi Code"
description: "Set up Kimi Code AI coding assistant"
icon: /logos/kimi-logo.svg
---

<Steps>
<Step title="Run Kimi Code">
  ```bash
  npx -y @anthropic-ai/kimi-code
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted. For more details, see the [official Kimi documentation](https://github.com/nicepkg/kimi-code).
</Step>

<Step title="Start agents-chatgroup">
  Once authenticated, launch agents-chatgroup:

  ```bash
  npx agents-chatgroup
  ```

  You can now select Kimi Code when creating task attempts.
</Step>
</Steps>
