<Steps>
<Step title="运行 Amp">
  ```bash
  npx -y @sourcegraph/amp
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。更多详情请参阅 [Amp 用户手册](https://ampcode.com/manual#install)。
</Step>

<Step title="启动 agents-chatgroup">
  认证完成后，启动 agents-chatgroup：

  ```bash
  npx agents-chatgroup
  ```

  或直接启动已安装的桌面应用。安装方式请参考[快速开始](/zh-Hans/getting-started)。

  现在您可以在创建任务尝试时选择 Amp。
</Step>
</Steps>
