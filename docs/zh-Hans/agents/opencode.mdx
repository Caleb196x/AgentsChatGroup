---
title: "OpenCode"
description: "设置 SST 的 OpenCode"
icon: /logos/opencode.svg
---

<Steps>
<Step title="运行 OpenCode">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。更多详情请参阅 [OpenCode GitHub 页面](https://github.com/sst/opencode)。
</Step>

<Step title="启动 agents-chatgroup">
  认证完成后，启动 agents-chatgroup：

  ```bash
  npx agents-chatgroup
  ```

  现在您可以在创建任务尝试时选择 OpenCode。
</Step>
</Steps>
