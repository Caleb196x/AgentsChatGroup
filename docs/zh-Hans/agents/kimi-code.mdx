description: "设置 Kimi Code AI 编程助手"

---

<Steps>
<Step title="运行 Kimi Code">
  ```bash
  npx -y @anthropic-ai/kimi-code
  ```
</Step>

<Step title="按照登录说明操作">
  按照提示完成认证流程。更多详情请参阅 [Kimi 官方文档](https://github.com/nicepkg/kimi-code)。
</Step>

<Step title="启动 agents-chatgroup">
  认证完成后，启动 agents-chatgroup：

  ```bash
  npx agents-chatgroup
  ```

  现在您可以在创建任务尝试时选择 Kimi Code。
</Step>
</Steps>
