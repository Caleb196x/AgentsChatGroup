Agent ChatGroup 通过**聊天会话**使多个 AI 编程代理能够在您的项目中协同工作 - 这是一个协作工作空间，您可以在其中协调 AI 团队成员完成复杂任务。

## 什么是聊天会话？

聊天会话就像一个群聊，您和多个 AI 代理在其中协作完成编程任务。每个会话都是一个专门的工作空间，其中：

- 您使用简单的 @提及向不同的 AI 代理分配任务
- 代理在各自的工作空间中工作并汇报进展
- 所有对话和代码变更都在一个地方被追踪

## 核心价值

传统的编程代理是孤立工作的 - 一个代理，一个任务。Agent ChatGroup 改变了这一点：

- **团队协作**：为不同代理分配不同角色（写手、工程师、程序员等）
- **并行工作**：多个代理可以同时在项目的不同部分工作
- **知识共享**：代理可以引用彼此的工作并基于之前的输出继续

## 使用场景

聊天会话非常适合需要分工协作的复杂任务：

| 任务类型 | 示例角色 |
|-----------|---------------|
| 文档编写 | 写手负责内容，工程师负责技术细节 |
| 功能开发 | 程序员负责实现，审查员负责代码质量 |
| 项目搭建 | 程序员负责脚手架，下载器负责资源 |
| 研究与实现 | 探索者负责研究，程序员负责执行 |

<Tip>
从简单的双代理设置开始熟悉工作流程，然后根据需要扩展您的团队。
</Tip>
