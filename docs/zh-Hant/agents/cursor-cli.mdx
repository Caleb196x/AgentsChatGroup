---

<Steps>
<Step title="安裝 Cursor Agent CLI">
  ```bash
  curl https://cursor.com/install -fsS | bash
  ```

  使用 `cursor-agent --version` 驗證安裝。更多詳情請參閱 [官方安裝指南](https://docs.cursor.com/en/cli/installation)。
</Step>

<Step title="按照登入說明操作">
  使用 `cursor-agent login` 登入（開啟瀏覽器）。您也可以設定 `CURSOR_API_KEY` 環境變數。完整說明：https://docs.cursor.com/en/cli/reference/authentication
</Step>

<Step title="啟動 agents-chatgroup">
  認證完成後，啟動 agents-chatgroup：

  ```bash
  npx agents-chatgroup
  ```

  現在您可以在建立任務嘗試時選擇 Cursor Agent CLI。
</Step>
</Steps>
