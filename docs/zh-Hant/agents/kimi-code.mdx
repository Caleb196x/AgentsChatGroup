<Steps>
<Step title="執行 Kimi Code">
  ```bash
  npx -y @anthropic-ai/kimi-code
  ```
</Step>

<Step title="按照登入說明操作">
  按照提示完成認證流程。更多詳情請參閱 [Kimi 官方文件](https://github.com/nicepkg/kimi-code)。
</Step>

<Step title="啟動 agents-chatgroup">
  認證完成後，啟動 agents-chatgroup：

  ```bash
  npx agents-chatgroup
  ```

  現在您可以在建立任務嘗試時選擇 Kimi Code。
</Step>
</Steps>
