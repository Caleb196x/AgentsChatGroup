---

<Steps>
<Step title="執行 OpenCode">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="按照登入說明操作">
  按照提示完成認證流程。更多詳情請參閱 [OpenCode GitHub 頁面](https://github.com/sst/opencode)。
</Step>

<Step title="啟動 agents-chatgroup">
  認證完成後，啟動 agents-chatgroup：

  ```bash
  npx agents-chatgroup
  ```

  現在您可以在建立任務嘗試時選擇 OpenCode。
</Step>
</Steps>
