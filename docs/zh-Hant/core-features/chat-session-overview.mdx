Agent ChatGroup 透過**聊天會話**使多個 AI 程式設計代理能夠在您的專案中協同工作 - 這是一個協作工作空間，您可以在其中協調 AI 團隊成員完成複雜任務。

## 什麼是聊天會話？

聊天會話就像一個群聊，您和多個 AI 代理在其中協作完成程式設計任務。每個會話都是一個專門的工作空間，其中：

- 您使用簡單的 @提及向不同的 AI 代理分配任務
- 代理在各自的工作空間中工作並回報進展
- 所有對話和程式碼變更都在一個地方被追蹤

## 核心價值

傳統的程式設計代理是孤立工作的 - 一個代理，一個任務。Agent ChatGroup 改變了這一點：

- **團隊協作**：為不同代理分配不同角色（寫手、工程師、程式設計師等）
- **並行工作**：多個代理可以同時在專案的不同部分工作
- **知識共享**：代理可以引用彼此的工作並基於之前的輸出繼續

## 使用場景

聊天會話非常適合需要分工協作的複雜任務：

| 任務類型 | 範例角色 |
|-----------|---------------|
| 文件編寫 | 寫手負責內容，工程師負責技術細節 |
| 功能開發 | 程式設計師負責實作，審查員負責程式碼品質 |
| 專案搭建 | 程式設計師負責腳手架，下載器負責資源 |
| 研究與實作 | 探索者負責研究，程式設計師負責執行 |

<Tip>
從簡單的雙代理設定開始熟悉工作流程，然後根據需要擴展您的團隊。
</Tip>
