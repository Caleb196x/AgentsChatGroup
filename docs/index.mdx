## What is AgentsChatGroup?

**AgentsChatGroup** creatively introduces a **work group chat** mode, enabling multiple AI agents to collaborate efficiently like real team members.

- One project = One group chat
- Add any number of AI members with different roles
- Command all agents in the same conversation context
- Agents automatically collaborate and communicate with each other
- You act as the project leader, orchestrating your AI team
