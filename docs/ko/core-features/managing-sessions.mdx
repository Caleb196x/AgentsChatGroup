채팅 세션은 AI 협업 워크플로우의 기반입니다. 효과적으로 생성하고 관리하는 방법은 다음과 같습니다.

## 새 세션 생성

1. **채팅** 페이지(`/chat`)로 이동
2. 왼쪽 사이드바에서 **새 세션** 클릭
3. 새 세션이 생성되고 자동으로 해당 세션으로 리디렉션

<Tip>
각 새 세션은 비어있는 상태로 시작하며, AI 멤버를 추가하고 협업을 시작할 준비가 됩니다.
</Tip>

## 세션 제목 편집

상단의 세션 제목을 클릭하여 편집합니다. 나중에 세션을 식별하기 쉽도록 제목을 짧고 설명적으로(최대 20자) 유지하세요.

좋은 예:
- "문서 업데이트"
- "인증 기능"
- "버그 수정 #123"

## 세션 작업

### 세션 아카이브

프로젝트를 완료하거나 작업을 일시 중지할 때:

1. 아카이브하려는 세션 열기
2. **아카이브** 버튼 클릭
3. 세션이 읽기 전용이 됨

<Note>
아카이브된 세션은 모든 메시지와 AI 멤버를 보존하지만, 새 메시지를 보내거나 멤버를 수정할 수 없습니다.
</Note>

### 세션 복원

아카이브된 세션에서 작업을 계속하려면:

1. 목록에서 아카이브된 세션 찾기
2. **복원** 클릭
3. 세션이 다시 활성화됨

### 세션 삭제

세션을 영구적으로 삭제하려면:

1. 세션 열기
2. **삭제** 클릭
3. 삭제 확인

<Warning>
세션을 삭제하면 모든 메시지가 제거되며 되돌릴 수 없습니다.
</Warning>

## 세션 상태 이해

| 상태 | 설명 |
|-------|-------------|
| 활성 | 정상 작업 상태 - 메시지 전송 및 AI 멤버 관리 가능 |
| 아카이브됨 | 읽기 전용 상태 - 메시지와 멤버가 보존되지만 잠김 |
