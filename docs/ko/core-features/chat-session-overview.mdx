Agent ChatGroup은 **채팅 세션**을 통해 여러 AI 코딩 에이전트가 프로젝트에서 함께 작업할 수 있게 합니다 - AI 팀 멤버를 조율하여 복잡한 작업을 완료하는 협업 작업 공간입니다.

## 채팅 세션이란?

채팅 세션은 여러분과 여러 AI 에이전트가 코딩 작업에서 협업하는 그룹 채팅과 같습니다. 각 세션은 전용 작업 공간으로서:

- 간단한 @멘션으로 다른 AI 에이전트에게 작업 할당
- 에이전트들이 각자의 작업 공간에서 작업하고 보고
- 모든 대화와 코드 변경이 한 곳에서 추적

## 핵심 가치

전통적인 코딩 에이전트는 고립되어 작업합니다 - 하나의 에이전트, 하나의 작업. Agent ChatGroup은 이를 변경합니다:

- **팀 협업**: 다른 에이전트에게 다른 역할 할당 (라이터, 엔지니어, 코더 등)
- **병렬 작업**: 여러 에이전트가 프로젝트의 다른 부분에서 동시에 작업 가능
- **지식 공유**: 에이전트들이 서로의 작업을 참조하고 이전 출력을 기반으로 구축

## 사용 사례

채팅 세션은 분업이 유익한 복잡한 작업에 완벽합니다:

| 작업 유형 | 예시 역할 |
|-----------|---------------|
| 문서 작성 | 라이터가 콘텐츠 담당, 엔지니어가 기술 세부사항 담당 |
| 기능 개발 | 코더가 구현 담당, 리뷰어가 코드 품질 담당 |
| 프로젝트 설정 | 코더가 스캐폴딩 담당, 다운로더가 에셋 담당 |
| 연구 및 구현 | 탐색기가 연구 담당, 코더가 실행 담당 |

<Tip>
간단한 2에이전트 설정으로 시작하여 워크플로우에 익숙해진 후 필요에 따라 팀을 확장하세요.
</Tip>
