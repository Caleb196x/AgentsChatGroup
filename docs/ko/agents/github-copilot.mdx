<Steps>
<Step title="GitHub Copilot 실행">
  ```bash
  npx -y @github/copilot
  ```
</Step>

<Step title="로그인 지침 따르기">
  프롬프트가 표시되면 화면의 지침에 따라 `/login` 명령으로 인증합니다. 자세한 내용은 [GitHub Copilot CLI 문서](https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli)를 참조하세요.
</Step>

<Step title="agents-chatgroup 시작">
  인증 완료 후 agents-chatgroup 시작:

  ```bash
  npx agents-chatgroup
  ```

  이제 작업 시도를 생성할 때 GitHub Copilot을 선택할 수 있습니다.
</Step>
</Steps>
