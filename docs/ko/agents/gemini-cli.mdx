<Steps>
<Step title="Gemini CLI 실행">
  ```bash
  npx -y @google/gemini-cli
  ```
</Step>

<Step title="로그인 지침 따르기">
  프롬프트에 따라 인증 흐름을 완료합니다. 자세한 내용은 [geminicli.com](https://geminicli.com/)을 참조하세요.
</Step>

<Step title="agents-chatgroup 시작">
  인증 완료 후 agents-chatgroup 시작:

  ```bash
  npx agents-chatgroup
  ```

  이제 작업 시도를 생성할 때 Gemini CLI를 선택할 수 있습니다.
</Step>
</Steps>
