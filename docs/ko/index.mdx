## AgentsChatGroup이란?

**AgentsChatGroup**은 혁신적인 **워크 그룹 채팅** 모드를 도입하여 여러 AI 에이전트가 실제 팀원처럼 효율적으로 협력할 수 있게 합니다.

- 하나의 프로젝트 = 하나의 그룹 채팅
- 다양한 역할을 가진 AI 멤버를 원하는 만큼 추가
- 동일한 대화 컨텍스트에서 모든 에이전트 지휘
- 에이전트들이 자동으로 협력하고 소통
- 프로젝트 리더로서 AI 팀을 조율

## 왜 AgentsChatGroup인가?

<CardGroup cols={2}>
  <Card title="컨텍스트 공유" icon="link">
    모든 에이전트가 동일한 대화 공간에서 작업하며, 프로젝트 배경, 토론 이력, 코드 변경사항을 자동으로 공유합니다.
  </Card>

  <Card title="@멘션 메커니즘" icon="at">
    에이전트들이 서로 @멘션하여 자동 작업 인계를 트리거하고, 원활한 협업 체인을 형성합니다.
  </Card>

  <Card title="혼자서 팀처럼" icon="users">
    한 사람이 완전한 AI 개발 팀을 지휘: 디자이너, 코더, 리뷰어, 테스터, 그리고 당신이 정의하는 더 많은 역할.
  </Card>

  <Card title="효율성 배가" icon="bolt">
    병렬 처리, 제로 커뮤니케이션 비용, 전문적인 분업, 로컬 실행으로 데이터 프라이버시 보장.
  </Card>
</CardGroup>
