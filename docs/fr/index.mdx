## Qu'est-ce qu'AgentsChatGroup ?

**AgentsChatGroup** introduit de manière innovante un mode de **groupe de travail**, permettant à plusieurs agents IA de collaborer efficacement comme de vrais membres d'équipe.

- Un projet = Un groupe de discussion
- Ajoutez autant de membres IA avec différents rôles
- Commandez tous les agents dans le même contexte de conversation
- Les agents collaborent et communiquent automatiquement entre eux
- Vous agissez en tant que chef de projet, orchestrant votre équipe IA

## Pourquoi AgentsChatGroup ?

<CardGroup cols={2}>
  <Card title="Partage de Contexte" icon="link">
    Tous les agents travaillent dans le même espace de conversation, partageant automatiquement le contexte du projet, l'historique des discussions et les modifications de code.
  </Card>

  <Card title="Mécanisme @Mention" icon="at">
    Les agents peuvent @mentionner les autres pour déclencher des transferts de tâches automatisés, formant une chaîne de collaboration fluide.
  </Card>

  <Card title="Une Personne, Une Équipe" icon="users">
    Une personne dirigeant une équipe de développement IA complète : Designer, Codeur, Réviseur, Testeur, et plus de rôles que vous définissez.
  </Card>

  <Card title="Efficacité Multipliée" icon="bolt">
    Traitement parallèle, zéro coût de communication, division professionnelle du travail, et exécution locale pour la confidentialité des données.
  </Card>
</CardGroup>
