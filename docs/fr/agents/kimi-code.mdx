<Steps>
<Step title="Exécuter Kimi Code">
  Mac OS et Linux:
  ```bash
  # Linux / macOS
  curl -LsSf https://code.kimi.com/install.sh | bash
  kimi
  ```
  Windows:
  ```powershell
  # Windows (PowerShell)
  Invoke-RestMethod https://code.kimi.com/install.ps1 | Invoke-Expression
  kimi
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Complétez le flux d'authentification comme indiqué. Pour plus de détails, consultez la [documentation officielle de Kimi](https://moonshotai.github.io/kimi-cli/en/guides/getting-started.html).
</Step>

<Step title="Démarrer agents-chatgroup">
  Une fois authentifié, lancez agents-chatgroup :

  ```bash
  npx agents-chatgroup
  ```

  Vous pouvez maintenant sélectionner Kimi Code lors de la création de tentatives de tâches.
</Step>
</Steps>
