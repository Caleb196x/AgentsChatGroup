---
title: "Kimi Code"
description: "Configurer l'assistant de codage IA Kimi Code"
icon: /logos/kimi-logo.svg
---

<Steps>
<Step title="Exécuter Kimi Code">
  ```bash
  npx -y @anthropic-ai/kimi-code
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Complétez le flux d'authentification comme indiqué. Pour plus de détails, consultez la [documentation officielle de Kimi](https://github.com/nicepkg/kimi-code).
</Step>

<Step title="Démarrer agents-chatgroup">
  Une fois authentifié, lancez agents-chatgroup :

  ```bash
  npx agents-chatgroup
  ```

  Vous pouvez maintenant sélectionner Kimi Code lors de la création de tentatives de tâches.
</Step>
</Steps>
