---
title: "Qwen Code"
description: "Configurer l'assistant de code Qwen"
icon: /logos/qwen-logo.svg
---

<Steps>
<Step title="Exécuter Qwen Code">
  ```bash
  npx -y @qwen-code/qwen-code
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Complétez le flux d'authentification comme indiqué. Pour plus de détails, voir la [documentation officielle Qwen](https://github.com/QwenLM/qwen-code).
</Step>

<Step title="Démarrer agents-chatgroup">
  Une fois authentifié, lancez agents-chatgroup :

  ```bash
  npx agents-chatgroup
  ```

  Vous pouvez maintenant sélectionner Qwen Code lors de la création de tentatives de tâches.
</Step>
</Steps>
