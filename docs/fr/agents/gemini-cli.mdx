<Steps>
<Step title="Exécuter Gemini CLI">
  ```bash
  npx -y @google/gemini-cli
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Complétez le flux d'authentification comme indiqué. Pour plus de détails, voir [geminicli.com](https://geminicli.com/).
</Step>

<Step title="Démarrer agents-chatgroup">
  Une fois authentifié, lancez agents-chatgroup :

  ```bash
  npx agents-chatgroup
  ```

  Vous pouvez maintenant sélectionner Gemini CLI lors de la création de tentatives de tâches.
</Step>
</Steps>
