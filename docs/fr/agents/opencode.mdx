---

<Steps>
<Step title="Exécuter OpenCode">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Complétez le flux d'authentification comme indiqué. Pour plus de détails, voir la [page GitHub OpenCode](https://github.com/sst/opencode).
</Step>

<Step title="Démarrer agents-chatgroup">
  Une fois authentifié, lancez agents-chatgroup :

  ```bash
  npx agents-chatgroup
  ```

  Vous pouvez maintenant sélectionner OpenCode lors de la création de tentatives de tâches.
</Step>
</Steps>
