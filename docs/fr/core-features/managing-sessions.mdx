Les sessions de chat sont la base de votre flux de travail de collaboration IA. Voici comment les créer et les gérer efficacement.

## Créer une Nouvelle Session

1. Naviguez vers la page **Chat** (`/chat`)
2. Cliquez sur **Nouvelle Session** dans la barre latérale gauche
3. Une nouvelle session est créée et vous êtes automatiquement redirigé vers elle

<Tip>
Chaque nouvelle session commence vide, prête pour que vous ajoutiez des membres IA et commenciez à collaborer.
</Tip>

## Modifier le Titre de la Session

Cliquez sur le titre de la session en haut pour le modifier. Gardez les titres courts et descriptifs (maximum 20 caractères) pour vous aider à identifier les sessions plus tard.

Bons exemples :
- "Mise à jour Docs"
- "Fonctionnalité Auth"
- "Correction Bug #123"

## Actions de Session

### Archiver une Session

Lorsque vous terminez un projet ou souhaitez mettre le travail en pause :

1. Ouvrez la session que vous voulez archiver
2. Cliquez sur le bouton **Archiver**
3. La session devient en lecture seule

<Note>
Les sessions archivées préservent tous les messages et membres IA, mais vous ne pouvez pas envoyer de nouveaux messages ni modifier les membres.
</Note>

### Restaurer une Session

Pour continuer le travail sur une session archivée :

1. Trouvez la session archivée dans votre liste
2. Cliquez sur **Restaurer**
3. La session redevient active

### Supprimer une Session

Pour supprimer définitivement une session :

1. Ouvrez la session
2. Cliquez sur **Supprimer**
3. Confirmez la suppression

<Warning>
Supprimer une session supprime tous les messages et ne peut pas être annulé.
</Warning>

## Comprendre les États de Session

| État | Description |
|-------|-------------|
| Active | État de travail normal - vous pouvez envoyer des messages et gérer les membres IA |
| Archivée | État en lecture seule - les messages et membres sont préservés mais verrouillés |
