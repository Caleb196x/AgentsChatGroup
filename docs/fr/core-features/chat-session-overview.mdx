Agent ChatGroup permet à plusieurs agents de codage IA de travailler ensemble sur vos projets grâce aux **Sessions de Chat** - des espaces de travail collaboratifs où vous coordonnez les membres de l'équipe IA pour accomplir des tâches complexes.

## Qu'est-ce qu'une Session de Chat ?

Une Session de Chat est comme une discussion de groupe où vous et plusieurs agents IA collaborez sur des tâches de codage. Chaque session agit comme un espace de travail dédié où :

- Vous assignez des tâches à différents agents IA en utilisant de simples @mentions
- Les agents travaillent dans leurs propres espaces de travail et rapportent
- Toutes les conversations et modifications de code sont suivies en un seul endroit

## Valeur Centrale

Les agents de codage traditionnels travaillent isolément - un agent, une tâche. Agent ChatGroup change cela en permettant :

- **Collaboration d'Équipe** : Assignez différents rôles à différents agents (Rédacteur, Ingénieur, Codeur, etc.)
- **Travail Parallèle** : Plusieurs agents peuvent travailler simultanément sur différentes parties d'un projet
- **Partage de Connaissances** : Les agents peuvent référencer le travail des autres et construire sur les sorties précédentes

## Cas d'Utilisation

Les Sessions de Chat sont parfaites pour les tâches complexes qui bénéficient de la division du travail :

| Type de Tâche | Exemples de Rôles |
|-----------|---------------|
| Documentation | Rédacteur pour le contenu, Ingénieur pour les détails techniques |
| Développement de Fonctionnalité | Codeur pour l'implémentation, Réviseur pour la qualité du code |
| Configuration de Projet | Codeur pour le scaffolding, Téléchargeur pour les assets |
| Recherche & Implémentation | Explorateur pour la recherche, Codeur pour l'exécution |

<Tip>
Commencez avec une configuration simple à deux agents pour vous familiariser avec le flux de travail, puis élargissez votre équipe selon les besoins.
</Tip>
